To run the test on Emulator
1. Set up an emulator in Android Studio and install the Android apk
2. Get the device info using command adb devices in Terminal
3. Modify the Desired Capabilities accordingly
4. Run the appium server on http://127.0.0.1:4723 (Just type 'appium' in Terminal, it will run the appium server on default port)
5. Run the test compareList() in CompareList
